FIRESTORE SECURITY RULES FOR SERVICE BOOKING WIZARD
====================================================

Copy and paste these rules into your Firestore Security Rules editor:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Allow public read access to branches
    match /branches/{document=**} {
      allow read: if true;
    }
    
    // Allow public read access to services
    match /services/{document=**} {
      allow read: if true;
    }
    
    // Allow public read access to staff
    match /staff/{document=**} {
      allow read: if true;
    }
    
    // Allow public read access to bookings
    match /bookings/{document=**} {
      allow read: if true;
    }
    
    // Allow anyone to create bookings
    match /bookings/{document=**} {
      allow create: if true;
    }
    
    // Deny all other access by default
    match /{document=**} {
      allow read, write: if false;
    }
  }
}

NOTES:
------
1. These rules allow public read access to all booking data (branches, services, staff)
2. Anyone can create new bookings without authentication
3. For production, consider:
   - Adding authentication requirements
   - Restricting write access to authenticated users
   - Adding validation rules for booking data
   - Implementing rate limiting

EXAMPLE PRODUCTION RULES (with authentication):
================================================

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Allow public read access to branches, services, and staff
    match /branches/{document=**} {
      allow read: if true;
    }
    
    match /services/{document=**} {
      allow read: if true;
    }
    
    match /staff/{document=**} {
      allow read: if true;
    }
    
    // Allow authenticated users to create and read bookings
    match /bookings/{document=**} {
      allow create: if request.auth != null;
      allow read: if request.auth != null;
      allow update, delete: if request.auth != null && resource.data.userId == request.auth.uid;
    }
    
    // Deny all other access
    match /{document=**} {
      allow read, write: if false;
    }
  }
}
